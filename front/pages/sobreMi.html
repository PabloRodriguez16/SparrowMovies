Información sobre el proyecto

Tecnologías usadas

HTML

Html fue utilizada para crear la estructura general de la página web. Colocando etiquetas y asignandoles clases y IDs

CSS

CSS fue utilizado para darle estilos, colores posición y entre otras modificaciónes esteticas.

JS

JS fue utilizada para darle la lógica a esta página. Se crearon diferentes funciónes para darle interactividad a la página.

BOOTSTRAP

B

Bootstrap fue utilizada para darle estilos a la página de forma mas sencilla y comoda. Ademas, los elementos agregados mediante Bootstrap son responsive.

MONGODB

MongoDB fue utilizada para crear nuestro cluster y nuestra BDD. Esta se encarga,junto a la lógica de JavaScript, de almacenar las peliculas en la base de datos y mostrarlas en Home.

Esas tecnologías fueron utilizadas para este proyecto.
El conjunto de estas permite que aparezcan las películas en la página principal, que tengan su estructura y sus respectivos estilos.

Además de esta sección y la sección principal, hay una sección llamada "páginas recomendadas". En esa sección podemos ver las páginas mas utilizadas para ver series y películas!
